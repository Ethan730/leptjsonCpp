## 2018/11/21

完成了unicode的解析

注意的地方：如果字符串中本来有`'\0'`，初始化string的时候需要指定长度，即使用下面的构造函数。

```c++
string (const char* s, size_t n);
```

不然遇到`'\0'`初始化会停止，得到的string是不对的。
